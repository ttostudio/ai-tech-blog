## 概要

AI Tech Blog の記事詳細ページに「関連記事」セクションを追加。記事本文の直後に配置し、同じカテゴリの関連記事を最大3件表示する。

---

## 1. レイアウト仕様

### 1.1 配置

**記事詳細ページの構造**:
```
📄 [記事詳細] (article-detail)
├── 📋 article-header（タイトル、日付、著者等）
├── 📝 .content（記事本文）
├── ✨ Related Articles Section ← 【新規追加】
├── ← → Previous / Next Navigation （.article-nav）
└── footer
```

**関連記事セクション**は `.content` の直後、`.article-nav` の前に配置。

### 1.2 セクション内部構造

```
┌─────────────────────────────────────────────┐
│  関連記事                                      │ ← H2 見出し
├─────────────────────────────────────────────┤
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  │
│  │ Card 1   │  │ Card 2   │  │ Card 3   │  │ ← 3カラムグリッド (PC)
│  │          │  │          │  │          │  │
│  └──────────┘  └──────────┘  └──────────┘  │
└─────────────────────────────────────────────┘
```

## 2. コンポーネント仕様

### 2.1 見出し「関連記事」

- **要素**: `<h2>` タグ
- **テキスト**: 「関連記事」
- **フォントサイズ**: `--text-2xl` (1.5rem) / PC、`--text-lg` (1.125rem) / モバイル
- **フォントウェイト**: 700（bold）
- **カラー**: ダークテーム時 gradient (cyan → indigo)、ライトテーム時 gradient (teal → indigo)
  - `--color-gradient-start` から `--color-gradient-end` を使用
- **マージン**:
  - 上: 3rem / PC、2rem / モバイル
  - 下: 1.5rem
  - 左右: 0
- **ボーダー**: 下部 1px solid `--color-border`（パディング 0.5rem 0.5rem 0.5rem 0）
- **スタイル**: 既存の `.content :global(h2)` と同じグラデーション + bg-clip-text

### 2.2 グリッドコンテナ

- **要素**: `<div class="related-articles-grid">`
- **コンテナ型**: CSS Grid
- **レスポンシブ列数**:
  - **PC (≥1200px)**: 3 カラム（等幅）
  - **タブレット横 (768px～1199px)**: 2 カラム（等幅）
  - **タブレット縦・スマホ (<768px)**: 1 カラム（フル幅）
- **ギャップ**:
  - 列間: `--space-5` (1.25rem) / PC、`--space-4` (1rem) / モバイル
  - 行間: `--space-5` (1.25rem)
- **パディング**: 0（外側 margin で制御）

### 2.3 記事カード

- **コンポーネント**: 既存の `ArticleCard.astro` を再利用
- **Props パス**:
  - `title`: 記事タイトル
  - `slug`: 記事スラッグ（リンク先）
  - `excerpt`: 記事説明
  - `category`: カテゴリ（カード側で表示）
  - `tags`: タグ配列
  - `author`: 著者名
  - `publishedAt`: 公開日時
  - `thumbnailUrl`: サムネイル画像 URL（オプション）
  - `featured`: false（固定 — featured カード として表示しない）
  - `contentLength`: 記事本文字数（オプション）

- **カード仕様（既存 ArticleCard）**:
  - **高さ**: 可変（コンテンツで自動調整）
  - **カバー高**: 180px / PC、140px / モバイル（thumbnail variant）
  - **ホバーアニメーション**: translateY(-4px) + シャドウ強化（既存）
  - **テーマ対応**: `--color-card-*` トークンで自動的にダーク/ライト両対応

## 4. ダーク/ライトテーム対応

### 4.1 見出し

**ダークテーム** (`data-theme` 未指定 / `"dark"`):
- グラデーション: `#22d3ee` (cyan) → `#818cf8` (indigo)
- テキストカラー: 自動的に白へ背景クリップで反映

**ライトテーム** (`data-theme="light"`):
- グラデーション: `#0891b2` (teal) → `#6366f1` (indigo)
- テキストカラー: 自動的に背景クリップで反映

### 4.2 グリッドコンテナ・カード

- **カード**: `ArticleCard.astro` が既に `var(--color-card-*)` トークンで対応
- **背景**: `--color-card-bg` （自動的にテーマに応じて変化）
- **ボーダー**: `--color-card-border` （自動的にテーマに応じて変化）
- **シャドウ**: `--shadow-card`、ホバー時 `--shadow-card-hover` （自動的にテーマに応じて変化）

**追加 CSS は不要** — 既存トークンで完全に自動対応

## 5. アニメーション

### 5.1 初期表示

- **種類**: Fade + Slide Up
- **既存キーフレーム**: `ArticleCard.astro` の `@keyframes fade-slide-up` を再利用
- **アニメーション**:
  - 各カードが順序付きで上から下へスライド + フェードイン
  - 遅延: `calc(var(--i, 0) * 0.1s)` （第1カード 0ms、第2カード 100ms、第3カード 200ms）
- **継続時間**: 0.5s

### 5.2 ホバー（PC のみ）

- **種類**: Elevation + Border Color変化
- **変化**:
  - `transform: translateY(-4px)` （上に浮く）
  - `box-shadow: var(--shadow-card-hover)` （より強いシャドウ）
  - `border-color: rgba(34, 211, 238, 0.2)` （cyan accent のボーダー）
- **トランジション**: `var(--transition-base)` (200ms ease)
- **タッチデバイス**: ホバー効果なし（PC のみ有効化）

## 7. 実装上の考慮事項

### 7.1 データソース

- **取得元**: API からは同じカテゴリの記事を max=3 件で取得
- **クエリ**: 既存 `[slug].astro` の前後記事ロジック を参考に、同一カテゴリで limit=3 を適用
- **フォールバック**: 関連記事が0件の場合、セクション全体を非表示にする
- **除外**: 現在の記事（slug）を含めない

### 7.2 CSSモジュール化

- **スコープ**: `<style scoped>` で `[slug].astro` 内に定義
- **再利用**: `.related-articles-section`、`.related-articles-grid` の2つの主要クラスで制御
- **変数**: `--i` を使用したカード番号付きアニメーション遅延（既存 ArticleCard パターン）

### 7.3 パフォーマンス

- **画像**: ArticleCard が既に lazy loading 対応
- **レンダリング**: Grid による効率的なレイアウト（flex より軽い）
- **アニメーション**: GPU加速（`transform`、`opacity` のみ使用）

## 8. マークアップ例（構造）

```astro
{relatedArticles && relatedArticles.length > 0 && (
  <section class="related-articles-section">
    <h2>関連記事</h2>
    <div class="related-articles-grid">
      {relatedArticles.map((article, index) => (
        <div class="article-card-wrapper" style={`--i: ${index}`}>
          <ArticleCard {...article} />
        </div>
      ))}
    </div>
  </section>
)}
```

**注**:
- `--i` CSS 変数で各カード の animation-delay を制御
- `relatedArticles` は API 取得結果（max 3件）
